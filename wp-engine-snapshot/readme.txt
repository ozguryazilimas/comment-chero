== Description ==

The WP Engine Snapshot plugin is used internally to migrate customers to [WP Engine](http://wpengine.com). It can also be used by any user to provide on demand file and database archives.

== Installation ==

1. Upload `wpengine-snapshot/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
